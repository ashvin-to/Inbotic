# Inbotic Frontend

This directory contains the React + Vite frontend for Inbotic.

## Commands

From this directory (`frontend/`):

```bash
npm install
npm run dev
```

Additional commands:

```bash
npm run build
npm run preview
npm run lint
```

## Notes

- This frontend is kept alongside the backend-rendered templates intentionally.
- API integration helpers are in `src/lib/api.js`.
- Authentication and theme state are managed under `src/context/`.
